# Operator Manual

## Get initial passwords

In the **tools** directory, we have a few script for getting the intial passwords to login to the site.

The most used script is the argocd-password. From root direcotry, run the following command to get the prod profile password:

```
export KUBECONFIG=./metal/kubeconfig.prod.yaml
./tools/argocd-password
```
